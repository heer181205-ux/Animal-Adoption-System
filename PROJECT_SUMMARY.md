## 🚀 How to Run

1. **Start the development server**:
   ```bash
   npm start
   ```

2. **Open in browser**:
   Navigate to `http://localhost:3000`

3. **Explore the website**:
   - Browse animals
   - View animal details
   - Fill out adoption forms
   - Check the About and Contact pages

## 📱 Pages & Routes

| Route | Page | Description |
|-------|------|-------------|
| `/` | Home | Landing page with featured animals |
| `/animals` | Animals | Browse all available pets |
| `/animal/:id` | Animal Detail | Individual animal information |
| `/adopt/:id` | Adoption Form | Apply to adopt a specific pet |
| `/about` | About Us | Organization information |
| `/contact` | Contact | Contact form and info |
